Configuração do GitHub
Para publicar automaticamente, preencha os campos abaixo com suas credenciais do GitHub.
Os dados ficam salvos no seu navegador e nunca são enviados a ninguém — só são usados para subir o arquivo no seu repositório.
Como gerar o token: 1. Acesse github.com → clique no seu avatar → Settings
2. Role até o final → Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Nome: sg-dashboard | Expiration: No expiration
6. Marque apenas: ✅ repo
7. Clique Generate token e cole aqui
Publicar para o gestor
Ao clicar em Publicar agora, os dados atuais do painel são enviados para o GitHub.
O Netlify detecta a mudança automaticamente e o gestor vê o painel atualizado em segundos.
